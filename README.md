# graph-writer
This program should write graph of assigned function and do something other.

## Using
- 1 or 1.2 or 1,2 is numbers
- 1 + 2
- 1 - 2
- 1 * 2
- 1 / 2
- 1 // 2
- 2x
- ( )
- 2(...) or (...)2 means 2 * (...) or (...) * 2
- 2 ^ 10 means raising 2 to power 10
- 3 % 2 means 3 mod 2
- |2| means absolute value of 2
- log3(9) means logarithm base 3 of 9
